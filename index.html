מפת ממשקי API ממשלתיים

פרויקט של רם אגמוןהמידע באתר עלול להיות שגויהירוקים הוד השרון
הדף נפתח כקובץ מקומי (file://) ולכן אינו יכול לטעון את הנתונים.

הדפדפן חוסם מודולי ES ו-fetch ממקור null — זו הגבלת אבטחה של הדפדפן, לא תקלה באתר. הפעל שרת מקומי מתיקיית הפרויקט: 

python3 -m http.server 8000

ולאחר מכן פתח http://localhost:8000.

לחלופין, ללא שרת כלל — dist/map.html הוא קובץ יחיד ועצמאי שעובד בפתיחה ישירה מהדיסק, כולל הרצת בקשות חיות.

ממשקי API של גופים ממשלתיים בישראל. כל שדה נבדק ישירות מול השרת. CORS הוא השדה הקובע — בלעדיו לא ניתן לקרוא ל-API מדף סטטי בדפדפן.
